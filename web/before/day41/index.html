一个标准的HTML文件需要使用<!DOCTYPE html> 来声明

示例:一个HTML文件Demo


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
            <title>HTML学习</title>
    </head>
    <body>
        <code>一个标准的HTML文件需要使用<!DOCTYPE html> 来声明</code>
            <span>示例:一个HTML文件Demo</span>
    </body>
</html>
